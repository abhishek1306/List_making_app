# List_making_app
In this project you can make a list like : camping ,todo and grocery.

## Installation
step 1:
open cmd and clone repository using
```bash
$ git clone 
```
step 2:
run npm command for node modules
```bash
$ npm install
```

## Run
step 3:
```bash
$ npm start
```

step 4:
open local host in chrome browser 
```bash
http://localhost:3000
```
